AI Stack — Command Reference

// UFO³ · OpenCode · Hermes · n8n

🖥
UFO³
Windows robot — controls any app on screen
⚡ Windows PowerShell (as Admin)
Step 1 — start UFO (run this once)
cd C:\Windows\system32\UFO; py -3.11 -m ufo
Step 2 — when UFO asks "Your request:" type one of these
open notepad and type hello world
open excel and create a budget table
open word and write a business proposal for a coffee shop
open Discord and send a message to [name] saying hello
search google for AI news today
organise my downloads folder by file type
What it does
Clicks buttons in any Windows app
Types text into any app automatically
Takes screenshots to see your screen
Best for apps with no API (old software, desktop apps)
💻
OpenCode
AI developer — writes and fixes code
⚡ Ubuntu terminal (WSL)
Step 1 — start OpenCode
export PATH="$PATH:/home/aleko/.hermes/node/bin" && opencode
Step 2 — type one of these inside opencode
build me a simple portfolio website
fix all the bugs in this file
build a chrome extension that blocks youtube
write a python script that renames all my files
explain what this code does line by line
What it does
Builds entire apps from scratch
Fixes broken code automatically
Writes in any programming language
Reads and understands your codebase
🤖
Hermes
Local AI assistant — research and tasks
⚡ Ubuntu terminal (WSL)
Step 1 — start Hermes
source ~/.bashrc && hermes
Step 2 — type one of these inside hermes
search the web for top 10 AI tools in 2026 and write a report
find all PDF files on my computer and list them
write a business plan for a coffee shop
research my competitors and compare their prices
summarise this YouTube video: [paste URL here]
What it does
Browses the web and researches anything
Searches and manages files on your PC
Runs 90+ built-in skills
Works 100% locally and privately
⚙
n8n
The orchestrator — runs workflows automatically
⚡ Ubuntu terminal → then browser
Step 1 — start n8n
docker run -it --rm --name n8n -p 5678:5678 -v n8n_data:/home/node/.n8n docker.n8n.io/n8nio/n8n
Step 2 — open this in your browser
http://localhost:5678
What it does
Automates emails, files, tasks 24/7
Connects 400+ apps together
Runs on a schedule automatically
Triggers UFO, Hermes and OpenCode
Works while your PC is on
